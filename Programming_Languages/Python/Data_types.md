# Python datatypes

The core data-types are as follows:

- str
- bool
- float
- double
- ...

## Converting data-types

For every data-type there is a corresponding converter method, e.g:

```python
a_string_int = "32"
as_int = int(a_string_int)
# 32
a_float_int = "32.2"
as_float = float(a_float_int)
# 32.2

a_bool = "true"
as_bool = bool(a_bool)
# True
```
